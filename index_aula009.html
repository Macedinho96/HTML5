Lista não ordenada

Para esse tipo de lista utilizaremos a tag ul

  • Papel higiênico
  • Sabonete
  • Pasta de dente
  • Água sanitária

Da mesma forma que na lista ordenada temos algumas opções de marcadores, basta apenas colocar um type nas frete do ul que já aparecerá as opções disponíveis.

  • Papel higiênico
  • Sabonete
  • Pasta de dente
  • Água sanitária

Os tipos disponiveis são circle, square e disc

Lista de definições

Com esse recurso, podemos fazer uma lista de objetos e suas respectivas definições utilizando as tags dt(nome do Objeto) e dd (definição do objeto)

< ol >
Tag utilizada para criar uma lista ordenada de itens
< ul >
Tag utilizada para criar uma lista não ordenada de itens
< li >
Tag utilizada para adicionar itens dentro de uma lista ordenada ou não

exemplo de como fica a estrutura do código

        <ol>
            <li> Lorem ipsum </li>
            <li> Lorem ipsum </li>
            <li> Lorem ipsum </li>
            <li> Lorem ipsum </li>
                 
        </ol>

        <ul>
            <li> Lorem ipsum </li>

            <li> Lorem ipsum </li>
            <li> Lorem ipsum </li>   
        </ul>
